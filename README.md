# 🧠 AI-Powered Tutor

> *An intelligent, agentic application designed to transform the way learners interact with knowledge.*

**EduAI Tutor** is a personalized learning platform powered by advanced multi-agent AI orchestration. It adapts to each learner's pace, style, and goals—acting as an intelligent mentor that explains concepts, generates quizzes, and tracks progress in real-time.

---

## 🌟 Key Features

### 💬 **AI Tutor Agent**
- Expert explanations tailored to your difficulty level
- Real-world examples and analogies for clarity
- Multi-turn conversation with chat history
- Follow-up questions and deep dives into any concept
- Supports diverse subjects: Mathematics, Science, Computer Science, History, Literature

### 📝 **Quiz Generator Agent**
- Auto-generates custom multiple-choice questions
- Adaptive difficulty based on your learning level
- Instant feedback with detailed explanations
- Configurable quiz length (3, 5, or 10 questions)
- Automatic strong/weak area detection

### 📊 **Progress Tracker Agent**
- Real-time performance analytics with KPIs
- Tracks accuracy, sessions, and concepts learned
- Identifies weak and strong learning areas
- AI-powered personalized feedback & recommendations
- Suggests adaptive learning paths based on performance
